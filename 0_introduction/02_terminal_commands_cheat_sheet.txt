
Cd (Change directory into your project folder)
$ cd
$ cd Desktop
$ cd words_numbers_and_lists

Print working directory
$ pwd

List Files
$ ls

Go back a directory (if needed)
$ cd ..

Open VS Code, Josh Prock Method (don't forget the period)
Follow the prompts! The installation tips are helpful
$ code .

Installing the virtual environment (Once when you create)
$ virtualenv -p python3 venv

Activate your virtual Environment (Everytime you open your computer to develop!)
VS Code this automatically
$ . venv/bin/activate

Installing numpy package
$ pip install numpy

Installing pandas package
$ pip install pandas

Running hello world
$ python hello_world.py

Deactivate your virtual Environment
$ deactivate
